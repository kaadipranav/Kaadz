## Tech Stack

- **Next.js 14** (App Router)
- **TypeScript**
- **Tailwind CSS**
- **Framer Motion** (animations)
- **Lucide React** (icons)

## Getting Started

```bash
# Install dependencies
npm install

# Run development server
npm run dev

# Build for production
npm run build

# Start production server
npm start
```

Open [http://localhost:3000](http://localhost:3000) to view the site.

## Features

- Dark, immersive design with animated backgrounds
- Control room aesthetic
- GitHub stats integration
- Motion-heavy but clean UI
- Modular structure for easy expansion

## Structure

- `app/` - Next.js app directory
- `components/` - React components
  - `Hero` - Landing section
  - `TheLab` - Product showcase
  - `Systems` - Tools and methods
  - `CurrentlyBuilding` - Active work
  - `About` - Minimal about section
  - `GitHubStats` - GitHub integration
  - `AnimatedBackground` - Particle animation
